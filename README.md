# Statistical_Signal_Processing-SSP
中国科学院大学电子学院专业课-统计信号处理课程资料-黄志蓓老师

本资料包括课程PPT、我自己的平时作业答案、大作业(当然这个大作业我自己觉得写的就是为了凑合的，也没有很用心，借鉴价值不大，GPT都比我写得好)

这门课的考试题目就是平时作业题目或者简化，除了最后一题，但是也不难，老师考试没有为难大家

我记忆中考试题有：
第一题：线性估计，
第二题：充分估计量，
